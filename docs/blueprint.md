# **App Name**: VideoFlow Insights

## Core Features:

- Session Initialization: Landing screen to input participant ID and start the experiment.
- Controlled Video Feed: Displays preloaded short-form videos in a TikTok-style vertical scrolling interface, categorized by genre.
- Engagement Tracker: Automatically tracks view time, skips, likes, and comment interactions for each video.
- Experiment Timer: Displays a subtle countdown timer to track session duration.
- Session Data Recorder: Stores session data locally including participant ID, video ID, content genre, watch time, skips, and interaction type.
- Data Exporter: Allows secure export of session data to an admin interface.
- Database Integration: Automatically exports session data to a database for persistent storage and analysis.

## Style Guidelines:

- Primary color: Light blue (#B0E2FF) for a neutral, unobtrusive feel, mimicking TikTok's color scheme.
- Background color: Off-white (#F8F8FF), nearly desaturated, for minimal distraction, consistent with TikTok's design.
- Accent color: Pale green (#98FB98), slightly brighter than the primary, for interactive elements, similar to TikTok's interactive cues.
- Body and headline font: 'Inter' sans-serif for clean, readable text. This font works well for both headlines and body text in this context. Follows TikTok's modern font choice.
- Fullscreen mode to eliminate distractions; large touch targets for easy interaction. Exact replication of TikTok's layout.
- Simple, universal icons for likes, comments, and other interactive elements. Mimics TikTok's icon designs.
- Subtle animations for video transitions and user interactions, to give clear feedback without being distracting. Replicates TikTok's animation style.
